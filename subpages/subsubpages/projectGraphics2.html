Description:

A hybrid neural path tracer combining GPU acceleration (CUDA) with neural network inference and hardware acceleration (OptiX) for improved rendering performance and quality. This project is currently under research and development, and the source code will be shared after publication.

Responsibilities:
  • Implemented both homogeneous and heterogeneous volume feature using NanoVDB library support for sparse volume data.
  • Implemented MIS support for surface, volume, and environment map sampling for both local and global light sources.
  • Implemented thread compaction for optimizing thread activation.
  • Implemented neural inference support for thesis research.
  • Implemented hardware-accelerated path tracing support using NVIDIA OptiX.
  • Conducted master thesis to optimize heterogeneous volume rendering pipeline for performance improvements.

Implemented Features
  • Homogeneous and Heterogeneous Volumes

    • The implementation utilizes delta tracking approach for unbiased volume rendering result.



  • Multiple Importance Sampling (MIS)

    • MIS helps adapting the lighting scenario for both local and distant lighting components.



  • Stream Splitting and compaction

    • Stream separation between surface and volume buffer to collect and maintain ease of access to specific ray buffers for further implementation on neural approach.
