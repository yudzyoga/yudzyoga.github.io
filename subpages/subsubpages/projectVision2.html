Description:

As part of the final project of **3D Computer Vision** (3DCV final project) course at Saarland University, I designed an incremental bundle adjustment method. It automatically estimates both the camera extrinsics and 3D point clouds simultaneously using Least Square method. This method optimizes the parameters using Gauss-Newton optimization method with gradient descent (Powellâ€™s hybrid) and Soft-l1 loss function. Initially, it takes nearest image with high number of correspondences and extract its pose using essential matrix method. It then optimizes the 3D points and camera extrinsic using Bundle Adjustment and incrementally adds up number of cameras until everything converged as showed in the video.
