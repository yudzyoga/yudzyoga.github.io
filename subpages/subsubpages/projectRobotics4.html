Short Description:

The telepresence robot is a computer, tablet, or smartphone-controlled robot with a camera, screen, speaker, and microphone so that anyone interacting with it can see and hear what the robot is "seeing" and "hearing" at the same time.

Description:

A telepresence robot project to simulate the team’s capability in utilizing various robotics research. This project also provides the research capabilities of robot positioning using external encoders. This robot still presented in various conferences until today.

Responsibilities:
  • Lead and advised the mechanical development of a telepresence robot for the robotics team’s side project.
  • Manage and ensure the manufacturing batch and schedule with multiple vendors.
  • Working closely with another division (electrical and programming) to create breakthrough features.
  • The general development tasks include design, manufacture, assembly, and analysis for each part of the robot.
