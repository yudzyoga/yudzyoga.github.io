Repository link: pathtracer-opengl

Description:

My attempt on making a custom path tracer using OpenGL library from scratch, while also learning on managing OpenGL API calls.


Implemented features:
  • Application
  • Vertex and Index Buffer
  • Vertex Array (VAO)
  • Vertex Shader (GLSL)
  • Model-View-Projection
  • Fragment Shader (GLSL)
  • Compute Shader (GLSL)
  • Texture Sampling
  • Screen Output
  • BSDF and emission Sampling
  • Megakernel rendering style
