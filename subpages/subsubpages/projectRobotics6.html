Short Description:

The prototype of the solenoid-powered engine using a piston-crank mechanism.

Description:

A solenoid engine prototype powered by copper armature and piston system to generate electrical power using a DC motor.

Responsibilities:
  • Lead the electrical circuit development.
  • Manufacture and assembly the electrical system.
  • Develop the program to calculate the electrical power and visualize in a LCD interface.

Research topic
  • Control System

    A closed-loop control system developed to properly adjust the target speed and keep the constant velocity value with 3 speed mode (slow, medium, and fast).

  • Electrical and Visualization system

    The solenoid control system assembled using a measured and controlled relay system and visualized through an LCD system.
