Description:

This project is also a part of the **3D Computer Vision final project**, which uses Visual Odometry to a sequence of rgb and depth images to get the overall camera movement and entire poses through the scene. In order to get the pose, the image features extracted using SIFT feature extractor with FLANN feature matcher. Pose processing method are conducted using PnP (Point-n Projection) method which minimizes reprojection error between two frame sequence, resulted in a smooth and nice transition across the entire scene.
