Short Description:

A wearable device that translates disabled people's hand signs into voice.

Description:

The goal of this project is to improve communication by developing a tool that can recognize user hand gestures. The device has a gyroscope and an accelerometer built into a single MPU6050 module. The sensor is applied to each finger to determine its orientation. The data from each of these sensors will be analyzed by an Arduino Pro Micro microcontroller, which will then choose a gesture from the gesture library that has been established. The speaker will then pronounce the letters or words, allowing for easier communication.

Responsibilities:
  • Develop the electrical circuits and the recognition program.
  • Develop the text-to-speech system.
