Short Description:

The PID (Proportional-Integral-Derivative) control simulator uses a ball and plate to simulate each control mode.

Description:

A ball-and-plate system to simulate the PID control using the ball rolling on a rotating platform. This method simulates the effect of each control system, where the closed loop control system respons on the error produced and resulted in the rotated platform system.

Responsibilities:
  • Lead the product development.
  • Design, simulate, manufacture the mechanical aspect of the platform.
  • Design the circuit system and develop the all the code for the platform (inverse kinematics, low-level control system, and UI)
  • Present the project at the annual mechanical engineering seminar.

Research Topics and Results
  • Sensory System

    The sensor configuration to detect the ball’s position in 2D using the laser arrays with laser and LDR sensor. The configuration of laser arrays enables the ball position system in 2D space.

  • PID Control

    A proportional-integral-derivative to control the orientation of the plane by giving the feedback of the error to produce the proper orientation.

  • Base Design

    The design of the base uses four Nema-17 stepper motor with X mode configuration. This arrangement provided the capability to control the plane’s orientation using each of the legs position.
