Short Description:

An image enhancement method to convert a low-light image into a bright image.


Description:

This project is part of High Level Computer Vision (HLCV) course, in which I modify an architecture that was meant for image segmentation into an image segmentation network instead. It uses U2-Net architecture which is a further developments of the original U-Net architecture. Upon modifying the network, I also modify the type of inputs (RGB and YCbCr), loss functions (MAE, MSE, SSIM, LPIPS), and image scales (100%, 75%, and 50%).
